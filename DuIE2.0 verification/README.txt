DuIE 2.0 samples verification 

Results generated using qwen3-235b-a22b had to be verified since the original language is in Chinese. As such, we randomly sampled 5 examples from the test set, translated them and the corresponding prediction outputs using DeepL and saved the files here for others to see. 

Based on the sanity check done here, we observed that the outputs were being generated as expected, i.e., the dataset itself did not deviate from the norm and the model processed the inputs as it would in english.
